✦ Documentación del proyecto ✦

NPCForge

Convierte a las personas de tu vida en fichas de personaje de RPG con análisis psicológico generado por IA

Gemini API Vanilla JS Sin dependencias localStorage

📖 Cómo usarlo

1

Consigue tu API Key de Gemini

Ve a aistudio.google.com, inicia sesión con tu cuenta de Google y haz clic en Get API Key → Create API Key. Es completamente gratis y no requiere tarjeta de crédito.

AIzaSy... (empieza siempre así)
2

Abre index.html en el navegador

No necesita servidor ni instalación. Doble clic en index.html y funciona directamente. Pega tu API Key en el campo de la izquierda — se guarda automáticamente en tu navegador.

3

Describe a una persona

Pon su nombre, la relación que tienes con ella, su hobby, trabajo, selecciona rasgos de personalidad y escribe una descripción libre. Cuanto más detalle, mejor ficha generará la IA.

4

Genera y guarda la ficha

Haz clic en ⚔ Generar ficha. En 3-5 segundos tendrás el análisis completo. Guárdalo en "Mi aldea" para acceder a él más tarde. Los datos se guardan localmente en tu navegador.

Frase característica

Al final todos acabamos siendo la versión mediocre de lo que soñamos ser.

🎭 Motivación oculta

Necesita ser reconocido como el más inteligente del grupo, aunque lo disfrace de humildad intelectual. En el fondo, tiene miedo de que su potencial real nunca se materialice.

💀 Punto débil

El éxito visible de sus contemporáneos le desestabiliza completamente. Cuando alguien a quien considera igual le supera, se cierra en banda durante semanas.

🔑 Cómo ganar su confianza

Muéstrate genuinamente interesado en sus ideas, no solo en sus conclusiones. Cita cosas que te dijo en conversaciones anteriores. Nunca compitas con él directamente — si ganas, pierde el interés; si pierdes, te pierde el respeto. Pídele consejo sobre algo en lo que realmente sea bueno.

📁 Estructura del proyecto

npcforge/
  index.html ← HTML principal (solo estructura)
  css/
    base.css ← Variables CSS, reset, tipografía
    layout.css ← Header, formulario, galería, botones
    sheet.css ← Ficha RPG resultado
  js/
    storage.js ← Guardar/cargar NPCs en localStorage
    gemini.js ← Llamadas a la API + prompt RPG
    app.js ← Controlador principal, eventos, render
  README.html ← Esta página
