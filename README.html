🛠 Tech Stack

🌐

HTML5 semántico

Estructura de la aplicación. Sin frameworks, sin templating. Solo etiquetas limpias con IDs funcionales.

index.html

🎨

CSS Variables

Sistema de diseño completo con variables. Grid y Flexbox nativos. Sin Bootstrap, sin Tailwind.

base.css · layout.css · sheet.css

⚡

Vanilla JavaScript

ES2022 puro. Módulos IIFE para encapsulación. Sin jQuery, sin React, sin nada extra.

app.js · gemini.js · storage.js

🤖

Google Gemini API

Modelo gemini-2.0-flash. Llamadas con fetch nativo. Prompt de ingeniería para output JSON estructurado.

generativelanguage.googleapis.com

💾

localStorage

Persistencia de NPCs y API Key directamente en el navegador. Sin base de datos, sin backend.

storage.js

Google Fonts

Cinzel (display/títulos) · Crimson Text (cuerpo) · VT323 (stats mono). Estética RPG medieval.

fonts.googleapis.com

📖 Cómo usarlo

1

Consigue tu API Key de Gemini

Ve a aistudio.google.com, inicia sesión con tu cuenta de Google y haz clic en Get API Key → Create API Key. Gratis, sin tarjeta de crédito. La key empieza siempre por AIza…

AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXX
2

Abre index.html en el navegador

Sin servidor, sin instalación, sin npm. Doble clic en index.html y funciona. La API Key se guarda automáticamente en localStorage para que no tengas que pegarla cada vez.

3

Describe a una persona de tu vida

Nombre, relación contigo, hobby, trabajo, rasgos de personalidad y una descripción libre. Cuanto más detalle escribas, más preciso y útil será el análisis. Puedes escribir desde dos frases hasta varios párrafos.

4

Genera la ficha y guárdala en tu aldea

Haz clic en ⚔ Generar ficha. En 3-5 segundos aparecerá el análisis completo. Usa el botón Guardar en la aldea para conservarlo. Los datos viven en tu navegador, nadie más los ve.
